UG Project - AGRIBOT

Aim: Design of an Autonomous Agricultural Robot - AGRIBOT

  • Advisor: Dr. Anand Darji
  • Worked on data acquisition system & implemented sensor fusion algorithms with IMU & Magnetometer data
  • Developed software architecture and simulated different algorithms of autonomous task in Gazebo simulator
  • Implemented DGPS concept using 2 different GPS sensors
  • Involved in structural design for robot, by URDF & SDF modeling for AGRIBOT and farm environment using XML
  • Developed prototype using Nvidia Jetson Nano, Arduino 2560, 10A Cytron Motor Driver, GPS(Ublox Neo M8N), PI Camera and MPU 9265
  • Project details can be found here: Github Link,Report,Presentation
  • Project got Funded by TEQIP - III Scheme of Govt of India - Proposoal

Mars Rover

Design of an Martion Rover

  • Advisor: Dr. Pinal J Engineer
  • Achieved live Video Streaming using Wi-Fi protocol of RPI 3B+ module with latency of 1 to 3 sec for remote functionalities over distance of 80 meters
  • Developed differential drive system for four & six wheel rover
  • Worked on over current and reverse polarity protection using MOSFET for power module
  • Interfaced sensors like IMU, GPS, DHT with Arduino for scientific data analysis
  • Electronics team lead (IRC - 2019) & Volunteered managerial head (IRC - 2020)
  • Project details can be found here: [Github Repository], [Report], [System Acceptance Video]

Research Project - Working with P3DX Robot

  • Used Kinect Xbox-360 to generate ambient environment and SLAM application
  • Developed URDF for Pioneer 3-DX using XML language

Maze Solver
